# **App Name**: LearnVerse

## Core Features:

- Role-Based Authentication: Secure login system for Students, Mentors, and Admins with role-specific permissions.
- Interactive Course Access: Browse, enroll, and access structured course modules with progress tracking for Students.
- Code Challenge Environment: Integrated code editor with real-time execution for coding practice challenges.
- Content Creation Tool: Mentors can create, edit, and manage courses and coding problems.
- Offline Video Access: Support for offline viewing of video lectures using PWA caching.
- Usage Analytics Dashboard: Admins monitor platform usage, track user activity, and manage roles with a visual dashboard.
- Intelligent Recommendation Tool: Leverage generative AI to offer course and mentor suggestions to users based on their activities and preferences. Generative AI will reason and incorporate the preferences and history for recommendations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and intelligence, aligning with educational content.
- Background color: Light gray (#F5F5F5) for a clean, distraction-free learning environment.
- Accent color: Vibrant orange (#FF9800) to highlight interactive elements and calls to action.
- Body text font: 'PT Sans', a humanist sans-serif, providing a modern look that is well suited to long text.
- Headline font: 'Space Grotesk', a geometric sans-serif with a contemporary, techy feel
- Use flat, minimalist icons for course categories and interactive elements.
- Clean and structured layout with clear sections for courses, coding practice, and mentorship.
- Subtle animations and transitions to enhance user engagement.
